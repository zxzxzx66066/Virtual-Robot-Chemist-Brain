1.克隆与环境准备
# cmd:终端运行下列代码,使用正确的仓库链接克隆
git clone https://github.com/zxzxzx66066/Virtual-Robot-Chemist-Brain.git

 # 进入项目文件夹
cd Virtual-Robot-Chemist-Brain

# 创建纯净的 Python 虚拟环境
python -m venv venv

# 激活虚拟环境
.\venv\Scripts\activate 

# 一键安装所有后端依赖
pip install fastapi uvicorn redis fakeredis pydantic zhipuai
pip install sniffio

2.配置模型秘钥
打开 Windows 的文件资源管理器，，进入你刚刚克隆代码的全新目录：C:\Users\用户名\Virtual-Robot-Chemist-Brain
在 main.py 中
client = ZhipuAI(api_key="*********") # 填入你的智谱 GLM-4 API Key并保存

3.启动调度中枢 (Backend)
cmd:终端运行下列代码
# 启动后端高并发网关
uvicorn main:app --reload 

4.启动控制台 (Frontend)
打开 Windows 的文件资源管理器，进入你刚刚克隆代码的目录：
C:\Users\用户名\Virtual-Robot-Chemist-Brain
直接使用浏览器打开项目根目录下的 index.html，即可体验全栈实时调度。
在页面的绿色输入框里，输入口语化学术指令。例如：“帮我把载玻片用机械臂送去载物台中心，然后再用显微镜对刚合成的纳米金颗粒做一次表面扫描。”
